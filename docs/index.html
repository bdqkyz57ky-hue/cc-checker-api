You are integrating with the BlackXCard CC Checker API.
Base URL: http://127.0.0.1:8000/api/v1

The API follows an OpenAI-like pattern with two main endpoints:

1. LIST GATES (Models)
   GET /gates/models
   Returns a list of available payment gateways.
   Response Format:
   {
     "object": "list",
     "data": [
       { "id": "braintree", "name": "Braintree", "type": "auth" },
       { "id": "paypal", "name": "PayPal", "type": "charge" },
       { "id": "shopify", "name": "Shopify", "type": "charge" },
       { "id": "stripe", "name": "Stripe", "type": "auth" },
       { "id": "sk", "name": "Stripe SK", "type": "charge" }
     ]
   }

2. CHECK CARD
   POST /gates/check
   Checks a credit card against a specific gate.
   
   Request Body:
   {
     "gate": "braintree",           // The ID of the gate to use
     "card": "4444444444444444|12|2028|123", // Full card string or just number
     "mm": "12",                    // Optional if passed in card string
     "yy": "2028",                  // Optional if passed in card string
     "cvc": "123",                  // Optional if passed in card string
     "site": "https://myshop.com"   // REQUIRED ONLY for 'shopify' gate
   }

   Response Format:
   {
     "gate": "braintree",
     "status": "approved" | "declined" | "error",
     "message": "Detailed response from gateway",
     "elapsed": 1.23,               // Time taken in seconds
     "details": { ... },            // Extra data like price, decline code
     "input_card": "..."
   }

IMPORTANT NOTES for AI Implementation:
- Always use the 'gate' parameter to select the gateway.
- For 'shopify' gate, you MUST ask the user for a target site URL.
- Handle 'approved' status as a success.
- 'elapsed' time is useful for monitoring performance.
- Use the LIST endpoint to dynamically discover available gates.

POST /gates/check

Perform a card check using a specified gateway.

Parameters

Field Type Required Description
gate string Yes The ID of the gate (e.g., braintree, stripe)
card string Yes Full card string or number
mm string No Expiration Month
yy string No Expiration Year
cvc string No Security Code
site string * Required ONLY for shopify gate

Example Request

{
  "gate": "stripe",
  "card": "4111111111111111|12|2026|123"
}

Example Response

{
  "gate": "stripe",
  "status": "approved",
  "message": "Payment successful",
  "elapsed": 2.45,
  "details": {},
  "input_card": "4111111111111111|12|2026|123"
}
